# Mitglieder bearbeiten

## So kannst du die Mitglieder auf deinem Grundstück bearbeiten: {id="plot-edit-members"}

1. Gebe `/protect` im Chat ein und klicke nun im angezeigten Menü auf **Meine Grundstücke** um dir
   alle deine Grundstücke anzeigen zu lassen. Hier findest du ebenfalls weitere Informationen zu deinem
   Grundstück, wie z. B. die Größe oder dessen Koordinaten.

   ![select my plots](plot-my-plot.png) {border-effect="rounded" thumbnail="false" width="384"}

2. Wähle nun das Grundstück aus, auf welchem du die Mitglieder bearbeiten möchtest.

   ![select plot](plot-select-plot.png) {border-effect="rounded" thumbnail="false" width="384"}
3. Klicke **links auf den Spielerkopf**.

   ![select member](plot-add-member.png) {border-effect="rounded" thumbnail="false" width="384"}

<tabs>
<tab title="Spieler zum Grundstück hinzufügen" id="plot-add-member">
Nun kannst du, wenn du den Block oben anklickst, einen Spieler zu deinem Grundstück hinzufügen. <br/>
Tippe den Spielernamen ein und klicke dann auf „Suchen“ um den Vorgang abzuschließen.

![add member menu](plot-add-member-menu.png) {border-effect="rounded" thumbnail="false" width="384"}
![add member menu](plot-search-for-player.png) {border-effect="rounded" thumbnail="false" width="384"}

<warning id="plot-member-add-info">

 - Achte darauf, dass der eingetippte Spielername den Voraussetzungen entspricht und keine
 Leerzeichen oder ungültige Zeichen enthält!
 - Der Spieler muss bereits einmal auf dem Server gespielt haben, damit er hinzugefügt werden kann.
</warning>

</tab>
<tab title="Spieler vom Grundstück entfernen" id="plot-remove-member">
Um einen Spieler zu entfernen drücke auf den Kopf des zu entfernenden Spielers und bestätige mit einem <shortcut>Linksklick</shortcut> das grüne Glas. 

![add member menu](plot-remove-member-confirmation.png) {border-effect="rounded" thumbnail="false" width="384"}
</tab>
</tabs>
