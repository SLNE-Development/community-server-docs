# Flags bearbeiten

## So kannst du deine Grundstück Flags anpassen: {id="plot-edit-flags"}

1. Gebe `/protect` im Chat ein und klicke nun im angezeigten Menü auf **Meine Grundstücke** um dir
   alle deine Grundstücke anzeigen zu lassen. Hier findest du ebenfalls weitere Informationen zu deinem
   Grundstück, wie z. B. die Größe oder dessen Koordinaten.
   ![select my plots](plot-my-plot.png) {border-effect="rounded" thumbnail="false" width="384"}

2. Wähle nun das Grundstück aus, auf welchem du die Flags bearbeiten möchtest.
   ![select plot](plot-select-plot.png) {border-effect="rounded" thumbnail="false" width="384"}

3. Klicke auf die **Redstockefackel**.
   ![select flags](plot-edit-flags.png) {border-effect="rounded" thumbnail="false" width="384"}

4. Nun kannst du die Flags deines ausgewählten Grundstückes anpassen.\
   Du kannst z. B. auswählen, ob andere Spieler auf deine Truhen Zugriff haben oder auf Tieren reiten können.
   ![claim](plot-flags.png) {border-effect="rounded" thumbnail="false" width="480" height="250"}

5. Um beispielsweise den Status für den Kistenzugriff auf dem Grundstück zu verändern, führe einen <shortcut>
   Linksklick</shortcut> auf die Kiste aus.
   ![plot-chest-access-members](plot-chest-access-members.png) {border-effect="rounded" thumbnail="false" width="480" height="250"}
   ![plot-chest-access-all](plot-chest-access-all.png) {border-effect="rounded" thumbnail="false" width="480" height="250"}

### Folgende Flags kannst du bearbeiten {collapsible="true" default-state="collapsed" id="plot-flags"}

| Flag-Name                    | Beschreibung                                                                                |
|:-----------------------------|:--------------------------------------------------------------------------------------------|
| **Kistenzugriff**            | Legt fest, ob fremde Spieler Inventare wie Kisten, Fässer oder Shulkerboxen öffnen dürfen.  |
| **Blockbenutzung**           | Legt fest, ob fremde Spieler interaktive Blöcke wie Türen, Hebel oder Knöpfe nutzen dürfen. |
| **Tiere verletzen**          | Legt fest, ob fremde Spieler Tiere verletzen dürfen.                                        |
| **Schlafen**                 | Legt fest, ob fremde Spieler in Betten schlafen dürfen.                                     |
| **Fahrzeuge platzieren**     | Legt fest, ob fremde Spieler Fahrzeuge wie bspw. Boote oder Loren platzieren dürfen.        |
| **Fahrzeuge zerstören**      | Legt fest, ob Fahrzeuge zerstört werden dürfen.                                             |
| **Reiten**                   | Legt fest, ob fremde Spieler auf Tieren oder in Fahrzeugen reiten dürfen.                   |
| **Gegenstandsrahmen drehen** | Legt fest, ob fremde Spieler Gegenstände in Rahmen drehen dürfen.                           |
| **Amboss benutzen**          | Legt fest, ob fremde Spieler einen Amboss benutzen dürfen.                                  |
| **Tropfblatt Nutzung**       | Legt fest, ob fremde Spieler große Tropfblätter auslösen dürfen.                            |
| **Explosionsschaden**        | Legt fest, ob Explosionen auf dem Grundstück Schaden verursachen dürfen.                    |
| **Enderman-Griefing**        | Legt fest, ob Endermänner Blöcke aufnehmen oder platzieren dürfen.                          |
| **Feuerausbreitung**         | Legt fest, ob sich Feuer auf dem Grundstück natürlich ausbreiten darf.                      |
| **Blattverfall**             | Legt fest, ob Blätter auf dem Grundstück natürlich verfallen dürfen.                        |
| **Graswachstum**             | Legt fest, ob sich Gras natürlich auf Erde ausbreiten darf.                                 |
| **Myzelwachstum**            | Legt fest, ob sich Myzel natürlich auf Erde ausbreiten darf.                                |
| **Pilzwachstum**             | Legt fest, ob Pilze auf dem Grundstück wachsen oder sich ausbreiten dürfen.                 |
| **Rankenwachstum**           | Legt fest, ob Ranken und Seetang wachsen oder sich ausbreiten dürfen.                       |
| **Tropfsteinwachstum**       | Legt fest, ob Tropfstein natürlich wachsen oder sich verlängern darf.                       |
| **Sculk-Ausbreitung**        | Legt fest, ob sich Sculk auf dem Grundstück ausbreiten oder wachsen darf.                   |
| **Pflanzenwachstum**         | Legt fest, ob Nutzpflanzen auf dem Grundstück wachsen dürfen.                               |
| **Korallenverblassen**       | Legt fest, ob Korallen ohne Wasser ausbleichen dürfen.                                      |
| **Schneegolem-Spuren**       | Legt fest, ob Schneegolems Schneespuren auf dem Boden hinterlassen dürfen.                  |
| **Schneefall**               | Legt fest, ob fallender Schnee Blöcke auf dem Grundstück bedecken darf.                     |
| **Schneeschmelze**           | Legt fest, ob Schnee durch Licht oder Wärme schmelzen darf.                                 |
| **Eisbildung**               | Legt fest, ob Wasser auf dem Grundstück zu Eis gefrieren darf.                              |
| **Eisschmelze**              | Legt fest, ob Eis durch Licht oder Wärme auf dem Grundstück schmelzen darf.                 |
