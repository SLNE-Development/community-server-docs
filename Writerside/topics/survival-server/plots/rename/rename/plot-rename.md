# Grundstück umbenennen

## So kannst du dein Grundstück umbenennen: {id="plot-rename"}

1. Gib `/protect` im Chat ein und klicke nun im angezeigten Menü auf **Meine Grundstücke**, um dir
   alle deine Grundstücke anzeigen zu lassen. Hier findest du ebenfalls weitere Informationen zu deinen
   Grundstücken, wie z. B. die Größe oder die Koordinaten deines Grundstücks.
   ![select my plots](plot-my-plot.png) {border-effect="rounded" thumbnail="false" width="480"}

2. Wähle nun das Grundstück aus, das du umbenennen möchtest.
   ![select plot](plot-select-plot.png) {border-effect="rounded" thumbnail="false" width="480"}

3. Klicke auf das **Namensschild**.
   ![select rename](plot-rename.png) {border-effect="rounded" thumbnail="false" width="480"}

4. Nun kannst du dein Grundstück beliebig umbenennen.\
   Klicke auf **Suchen**, um den Vorgang zu beenden.
   ![select name to rename](plot-rename-confirm.png) {border-effect="rounded" thumbnail="false" width="480"}

   > Beachte, dass der Name des Grundstücks nicht gegen das Regelwerk verstößt.
   >
   {style="warning"}
