# Shops

## 1. GRUNDLAGEN DES HANDELS
Der Handel auf dem Survival Server erfolgt zentral über **Shop-NPCs**, die an jedem Spawn zu finden sind. Ein Interagieren mit dem NPC öffnet das globale Handels-Interface.

### ITEM-INFORMATIONEN (HOVER-MENÜ)
Sobald du mit der Maus über ein ausgestelltes Item fährst, werden dir folgende Details angezeigt:
* **Verkaufspreis:** Kosten pro Item in CastCoins (CC).
* **Lagerbestand:** Wie viele Einheiten derzeit verfügbar sind.
* **Verkäufer:** Name des Spielers, dem der Shop gehört.
* **Zeitstempel:** Erstellungsdatum des Angebots.

### KAUFVORGANG
1.  Klicke das gewünschte Item im Inventar an.
2.  **Bestätigung:** Nutze den rechten Button im neuen Fenster.
3.  **Abbruch:** Nutze den linken Button, um zum Hauptmenü zurückzukehren.

## 2. EIGENEN SHOP ERSTELLEN
Um eigene Waren anzubieten, folge diesem Ablauf im Shop-Interface:

### SCHRITT 1: INITIALISIERUNG
Drücke am unteren Bildschirmrand auf den Knopf **"Shop erstellen"** (gekennzeichnet durch das Plus-Symbol).

### SCHRITT 2: ITEM-AUSEWAHL
* Klicke auf das Kopf-Symbol in der Mitte (**"Kein Item ausgewählt"**).
* Es öffnet sich dein persönliches Inventar auf der linken Seite.
* Wähle das Item aus, das du verkaufen möchtest.

### SCHRITT 3: PREISGESTALTUNG
Klicke auf den Button **"Preis pro Item festlegen"** direkt neben deinem gewählten Item.
* **Quick-Buttons:** Nutze **+1** oder **+50** zur schnellen Anpassung.
* **Manueller Preis:** Klicke oben auf **"Eigenen Preis eingeben"**, tippe den Betrag in den Dialog und bestätige diesen.

### SCHRITT 4: ABSCHLUSS
Bestätige die Erstellung mit dem Button **"Shop erstellen"**.

---

## 3. LAGERVERWALTUNG & WARTUNG
Ein neu erstellter Shop ist zunächst leer. Du musst Waren physisch in den Shop transferieren.

### ITEMS EINLAGERN
1.  Öffne dein Shop-Menü.
2.  Klicke auf das **Kisten-Symbol**.
3.  Wähle **"Items einlagern"**.
4.  Nutze `Shift + Klick`, um die Waren aus deinem Inventar in das Shop-Lager zu verschieben.

### SHOPS VERWALTEN & LÖSCHEN
* **Übersicht:** Klicke auf dein eigenes Kopf-Symbol im Menü, um deine aktiven Shops zu sehen.
* **Löschvorgang:** Halte `Shift` gedrückt und führe einen `Linksklick` auf den entsprechenden Shop aus. Bestätige im folgenden Dialog die Löschung.

---

## 4. SORTIERFUNKTIONEN
Nutze den **Comparator** (unten rechts), um die globale Ansicht nach deinen Wünschen zu filtern:

| Kriterium | Beschreibung |
| :--- | :--- |
| **Itemname** | Alphabetische Sortierung der Waren. |
| **Preis** | Sortierung aufsteigend oder absteigend. |
| **Zeit** | Sortierung nach Erstellungsdatum (neu/alt). |
| **Lagerstand** | Meist gelagerte Items werden zuerst angezeigt. |
| **Verkäufe** | Sortierung nach der Beliebtheit/Verkaufszahl. |

---

## 5. SICHERHEITSHINWEISE UND REGELN
* **Währung:** Der Handel erfolgt ausschließlich über **CastCoins (CC)**.
* **Wartungspflicht:** Verkäufer sind selbst dafür verantwortlich, ihre Bestände aufzufüllen. Leere Shops können nichts verkaufen.
* **Bug-Reporting:** Fehler im System (z.B. Duplizierungsfehler) müssen umgehend gemeldet werden.
* **Exploiting:** Das Ausnutzen von Fehlern wird als schwerer Regelverstoß geahndet.
